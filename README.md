# ai_agent

Daily-loop AI trading agent for US equities. Proposes buy/sell actions with
limit and stop levels via Telegram; orders execute through Trading 212 only
after explicit human approval.

V1 scope is intentionally narrow: US equities, free-tier market data, ~30-name
curated watchlist, paper account first. Crypto is a phase-2 deliverable.

See [`docs/spec.md`](docs/spec.md) for the full design.
